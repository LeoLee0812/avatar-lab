# 更新日志

## v1.0.1 — 2026-07-22

修菜单栏图标不显示的问题。

### 修复
- **菜单栏图标是空白的**：`MenuBarExtra` 用的 `person.crop.square.badge.magnifyingglass` 这个 SF Symbol 根本不存在，系统返回 nil 后画出来是一块看不见但能点的空白区域——app 在跑却像没装。换成 `person.crop.square`（SF Symbols 1.0 就有，macOS 13 必定可用）
- **存了图却像没存上**：图落在「桌面/AvatarLab/」子文件夹，而「已存到…」的提示写在浮窗底部，浮窗一失焦就关，提示根本来不及看。现在「存到桌面」存完直接在访达里选中该文件
- 「在访达显示」按钮和主按钮功能重复，换成「打开文件夹」，一键进 `桌面/AvatarLab/`
- 写盘失败时退到「存储为」面板：ad-hoc 签名每次重新编译代码标识都会变，系统给的桌面访问授权会作废，而无 Dock 图标的应用重新授权的弹窗未必抢得到焦点。面板走系统 powerbox，不受这个限制

### 其他
- 版本号收敛到 `build.sh` 里一处，`make_dmg.sh` 自动跟随，不会再出现 DMG 文件名和 app 内版本对不上

## v1.0.0 — 2026-07-22

首个版本。常驻菜单栏的头像小工具。

### 搜图（核心）
- 点开菜单栏自动随机搜一批头像候选，不用先想关键词
- 骰子按钮「换一批」：随机主题 + 随机页码；搜过词则翻下一页
- 六个主题一键切：随机 / 二次元 / 动物 / 风景 / 抽象 / 科技感
- Pexels + Pixabay 并发搜，结果交错混排，单家失败或限流不影响另一家
- Pexels 按 `orientation=square` 优先搜方图，结果不足再补一轮不限比例的
- 照片 / 插画切换（二次元、卡通走 Pixabay 的 illustration）
- 方图优先排序（不做硬过滤——实测一过滤经常只剩一两张）
- 内置约 90 个中文题材词表，中文关键词自动换成英文

### 裁切与导出
- 缩放 + 左右 / 上下焦点三个滑块，实时预览
- 可切圆形（透明角），导出 128 / 256 / 512 / 1024 px
- 回车存到 `桌面/AvatarLab/`，⌘C 复制到剪贴板

### AI 生成（可选）
- OpenAI 兼容 `/images/generations`，接口地址与模型名可配
- 12 个 1:1 头像风格预设

### 其他
- 无 Dock 图标（`LSUIElement`），密钥只存本机 UserDefaults
- `./build.sh` 编译打包，`./make_dmg.sh` 打 DMG
